## 3. Bottleneck 块结构

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              Bottleneck Block                                   │
│                         expansion = 4                                          │
└─────────────────────────────────────────────────────────────────────────────────┘

输入 x (通道: C_in)
       │
       ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           1×1 Conv (压缩)                                      │
│  Conv2d(C_in, C, 1×1) → BatchNorm2d → ReLU                                   │
│  通道变化: C_in → C                                                             │
└─────────────────────────────────────────────────────────────────────────────────┘
       │
       ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           3×3 Conv (空间特征提取)                              │
│  Conv2d(C, C, 3×3, stride) → BatchNorm2d → ReLU                             │
│  通道变化: C → C (stride 控制空间尺寸)                                         │
└─────────────────────────────────────────────────────────────────────────────────┘
       │
       ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           1×1 Conv (恢复)                                      │
│  Conv2d(C, C×4, 1×1) → BatchNorm2d                                          │
│  通道变化: C → C×4                                                             │
└─────────────────────────────────────────────────────────────────────────────────┘
       │
       ▼
       (+) ◄── 残差连接 (如果需要则通过 downsample 调整维度)
       │
       ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              ReLU 输出                                          │
│  输出通道: C × 4                                                               │
└─────────────────────────────────────────────────────────────────────────────────┘

参数数量: 1×1×C_in×C + 3×3×C×C + 1×1×C×(C×4) + 3×(C×4)
示例 (Layer1, C_in=256, C=64): ~70K 参数
```

└─────────────────────────────────────────────────────────────────────────────────┘

## 5. 锚框生成器

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              锚框生成器                                         │
│                        Anchor Generator                                         │
└─────────────────────────────────────────────────────────────────────────────────┘

基础配置:
- scales: [1.0, 1.2, 1.5]     (锚框面积缩放因子)
- ratios: [0.5, 1.0, 2.0]     (宽高比)
- strides: [8, 16, 32, 64]    (特征图相对原图的下采样倍数)

每个位置生成的锚框数量 = len(scales) × len(ratios) = 3 × 3 = 9

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           单个特征图锚框生成                                     │
└─────────────────────────────────────────────────────────────────────────────────┘

特征图尺寸: (H, W)
Stride: S
生成中心点: (S/2, S/2), (3S/2, S/2), ... (W*S - S/2, H*S - S/2)

每个中心点生成 9 个锚框:
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│   Scale=1.0, Ratio=0.5  →  锚框尺寸: (0.707S, 1.414S)  (瘦高)                 │
│   Scale=1.0, Ratio=1.0  →  锚框尺寸: (1.0S, 1.0S)     (正方形)               │
│   Scale=1.0, Ratio=2.0  →  锚框尺寸: (1.414S, 0.707S) (矮胖)                 │
│                                                                                 │
│   Scale=1.2, Ratio=0.5  →  锚框尺寸: (0.848S, 1.697S)                         │
│   Scale=1.2, Ratio=1.0  →  锚框尺寸: (1.2S, 1.2S)                              │
│   Scale=1.2, Ratio=2.0  →  锚框尺寸: (1.697S, 0.848S)                         │
│                                                                                 │
│   Scale=1.5, Ratio=0.5  →  锚框尺寸: (1.06S, 2.12S)                           │
│   Scale=1.5, Ratio=1.0  →  锚框尺寸: (1.5S, 1.5S)                             │
│   Scale=1.5, Ratio=2.0  →  锚框尺寸: (2.12S, 1.06S)                           │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

锚框坐标格式: [x1, y1, x2, y2] (左上角 + 右下角)

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           所有特征图锚框统计                                     │
│                                                                                 │
│  特征图      尺寸         锚框数        总锚框数                               │
│  ─────────────────────────────────────────────                               │
│  P3 (stride=8)   H/8 × W/8    9 × H/8 × W/8                                  │
│  P4 (stride=16)  H/16 × W/16  9 × H/16 × W/16                                │
│  P5 (stride=32)  H/32 × W/32  9 × H/32 × W/32                                │
│  P6 (stride=64)  H/64 × W/64  9 × H/64 × W/64                                │
│  ─────────────────────────────────────────────                               │
│  总计                                                    H×W × 1.33          │
│                                                                                 │
│  示例: 输入 640×640 图像 → 总锚框数 ≈ 54,300                                 │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 6. Head 结构详解

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           Classification Head                                  │
│                              分类头                                            │
└─────────────────────────────────────────────────────────────────────────────────┘

输入: P3/P4/P5/P6 特征图 (B, 256, H_i, W_i)
输出: 分类 logits (B, H_i × W_i × 9, Num_Classes)

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              单层结构                                           │
│                                                                                 │
│  输入特征图                                                                   │
│  (B, 256, H, W)                                                               │
│       │                                                                       │
│       ▼                                                                       │
│  ┌────────────────────────────────────────────────────────────────────────┐   │
│  │                    Shared Conv Block                                    │   │
│  │  Conv2d(256, 256, 3×3, padding=1) → BatchNorm → ReLU                 │   │
│  │  Conv2d(256, 256, 3×3, padding=1) → BatchNorm → ReLU                 │   │
│  └────────────────────────────────────────────────────────────────────────┘   │
│       │                                                                       │
│       ▼                                                                       │
│  ┌────────────────────────────────────────────────────────────────────────┐   │
│  │                    Classification Output                                │   │
│  │  Conv2d(256, 9×Num_Classes, 1×1)                                      │   │
│  │  输出: (B, 9×Num_Classes, H, W)                                        │   │
│  └────────────────────────────────────────────────────────────────────────┘   │
│       │                                                                       │
│       ▼                                                                       │
│  Reshape: (B, H×W×9, Num_Classes)                                             │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           Regression Head                                      │
│                              回归头                                            │
└─────────────────────────────────────────────────────────────────────────────────┘

输入: P3/P4/P5/P6 特征图 (B, 256, H_i, W_i)
输出: 边界框回归 (B, H_i × W_i × 9, 4)

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              单层结构                                           │
│                                                                                 │
│  输入特征图                                                                   │
│  (B, 256, H, W)                                                               │
│       │                                                                       │
│       ▼                                                                       │
│  ┌────────────────────────────────────────────────────────────────────────┐   │
│  │                    Shared Conv Block                                    │   │
│  │  Conv2d(256, 256, 3×3, padding=1) → BatchNorm → ReLU                   │   │
│  │  Conv2d(256, 256, 3×3, padding=1) → BatchNorm → ReLU                   │   │
│  └────────────────────────────────────────────────────────────────────────┘   │
│       │                                                                       │
│       ▼                                                                       │
│  ┌────────────────────────────────────────────────────────────────────────┐   │
│  │                    Regression Output                                     │   │
│  │  Conv2d(256, 9×4, 1×1)                                                 │   │
│  │  输出: (B, 9×4, H, W)                                                   │   │
│  └────────────────────────────────────────────────────────────────────────┘   │
│       │                                                                       │
│       ▼                                                                       │
│  Reshape: (B, H×W×9, 4)                                                       │
│                                                                                 │
│  4 个回归值: [dx, dy, dw, dh]                                                 │
│  - dx, dy: 中心点相对锚框中心的偏移量                                           │
│  - dw, dh: 宽高相对于锚框宽高的对数缩放                                         │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 7. 边界框编码与解码

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           边界框编码 (Encode)                                   │
│                    锚框 → 回归目标                                              │
└─────────────────────────────────────────────────────────────────────────────────┘

输入: 锚框 (x1, y1, x2, y2), GT 框 (gx1, gy1, gx2, gy2)
输出: 回归目标 (dx, dy, dw, dh)

计算过程:

1. 锚框参数:
   cx = (x1 + x2) / 2
   cy = (y1 + y2) / 2
   w = x2 - x1
   h = y2 - y1

2. GT 框参数:
   gcx = (gx1 + gx2) / 2
   gcy = (gy1 + gy2) / 2
   gw = gx2 - gx1
   gh = gy2 - gy1

3. 回归目标:
   dx = (gcx - cx) / w           ← 中心 x 偏移 (相对于锚框宽度)
   dy = (gcy - cy) / h           ← 中心 y 偏移 (相对于锚框高度)
   dw = log(gw / w)              ← 宽度缩放 (对数尺度)
   dh = log(gh / h)              ← 高度缩放 (对数尺度)

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           边界框解码 (Decode)                                   │
│                    锚框 + 回归预测 → 预测框                                      │
└─────────────────────────────────────────────────────────────────────────────────┘

输入: 锚框 (x1, y1, x2, y2), 回归预测 (dx, dy, dw, dh)
输出: 预测框 (px1, py1, px2, py2)

计算过程:

1. 锚框参数:
   cx = (x1 + x2) / 2
   cy = (y1 + y2) / 2
   w = x2 - x1
   h = y2 - y1

2. 预测中心:
   pcx = cx + dx × w
   pcy = cy + dy × h

3. 预测宽高:
   pw = w × exp(dw)
   ph = h × exp(dh)

4. 预测框:
   px1 = pcx - pw / 2
   py1 = pcy - ph / 2
   px2 = pcx + pw / 2
   py2 = pcy + ph / 2
```

## 8. 损失函数

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              完整损失函数                                       │
│                         Detection Loss                                         │
└─────────────────────────────────────────────────────────────────────────────────┘

总损失 = 分类损失 (α) + 回归损失 (β)

L = α × L_cls + β × L_reg

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           Focal Loss (分类损失)                                │
└─────────────────────────────────────────────────────────────────────────────────┘

L_cls = -α × (1 - p_t)^γ × log(p_t)

其中:
- p_t = p (如果正类) 或 1 - p (如果负类)
- α = 0.25 (平衡因子)
- γ = 2.0 (聚焦因子)

作用:
- 降低易分类样本的损失权重
- 聚焦于难分类的困难样本
- 解决类别不平衡问题

┌─────────────────────────────────────────────────────────────────────────────────┐
│                         Smooth L1 Loss (回归损失)                              │
└─────────────────────────────────────────────────────────────────────────────────┘

L_reg = SmoothL1(d_pred, d_target)

    ┌ |x| - 0.5,           |x| < 1
L = ┤
    └ 0.5 × x²,            |x| ≥ 1

优点:
- |x| < 1 时梯度稳定
- |x| ≥ 1 时梯度线性增长，避免梯度爆炸

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           正负样本匹配                                          │
└─────────────────────────────────────────────────────────────────────────────────┘

正样本条件:
- IoU(锚框, GT) ≥ 0.5 → 正样本，预测对应 GT 类别
- 每个 GT 框匹配 IoU 最高的锚框 → 正样本

负样本条件:
- IoU(锚框, GT) < 0.4 → 负样本，预测为背景类

忽略样本:
- 0.4 ≤ IoU(锚框, GT) < 0.5 → 忽略，不参与损失计算
```
