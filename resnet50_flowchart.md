## 2. ResNet-50 网络结构详解

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           ResNet-50 架构图                                  │
└─────────────────────────────────────────────────────────────────────────────┘

输入图像 (3, 32, 32)  ──────────────────────────────────────────────────────►

                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                              Stem 层                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Conv7x7, stride=2, padding=3  →  BatchNorm  →  ReLU  →  MaxPool    │   │
│  │  (3, 32, 32)                  (64, 16, 16)                           │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (64, 16, 16)

┌─────────────────────────────────────────────────────────────────────────────┐
│                           Stage 1: layer1                                   │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Bottleneck × 3                                                     │   │
│  │  ┌─────────────────────────────────────────────────────────────┐     │   │
│  │  │ 输入: 256 channels                                           │     │   │
│  │  │  ↓ 1x1 Conv (压缩) → 64 channels                            │     │   │
│  │  │  ↓ 3x3 Conv (处理) → 64 channels                            │     │   │
│  │  │  ↓ 1x1 Conv (恢复) → 256 channels                          │     │   │
│  │  │  ↓ + identity (残差连接)                                   │     │   │
│  │  │  ↓ ReLU                                                     │     │   │
│  │  └─────────────────────────────────────────────────────────────┘     │   │
│  │  (每个 block 输出: 256 channels)                                      │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (256, 16, 16)

┌─────────────────────────────────────────────────────────────────────────────┐
│                           Stage 2: layer2                                   │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Bottleneck × 4                                                     │   │
│  │  - 第一个 block: stride=2 (空间降维 16→8, 通道 256→512)              │   │
│  │  - 其余 3 个 block: stride=1                                        │   │
│  │  (输出: 512 channels)                                                │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (512, 8, 8)

┌─────────────────────────────────────────────────────────────────────────────┐
│                           Stage 3: layer3                                   │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Bottleneck × 6                                                     │   │
│  │  - 第一个 block: stride=2 (空间降维 8→4, 通道 512→1024)             │   │
│  │  - 其余 5 个 block: stride=1                                        │   │
│  │  (输出: 1024 channels)                                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (1024, 4, 4)

┌─────────────────────────────────────────────────────────────────────────────┐
│                           Stage 4: layer4                                   │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Bottleneck × 3                                                     │   │
│  │  - 第一个 block: stride=2 (空间降维 4→2, 通道 1024→2048)            │   │
│  │  - 其余 2 个 block: stride=1                                        │   │
│  │  (输出: 2048 channels)                                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (2048, 2, 2)

┌─────────────────────────────────────────────────────────────────────────────┐
│                              Head 层                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  AdaptiveAvgPool2d((1,1))  →  Flatten  →  Linear(2048, num_classes) │   │
│  │  (2048, 2, 2)              (2048)       (num_classes,)               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    ▼
                                    ▼  输出: (num_classes,)

                                    ▼
                              最终输出: 类别预测 logits
```

## 3. Bottleneck 块结构图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Bottleneck 残差块                                  │
└─────────────────────────────────────────────────────────────────────────────┘

                        输入 x ────────┬───────► (+) ──► ReLU ──► 输出
                                      │           ▲
                        ┌─────────────┴───────────┘
                        ▼
              ┌─────────────────────┐
              │    1x1 Conv, stride │   降维: in_channels → out_channels
              │    BatchNorm        │
              │    ReLU             │
              └──────────┬──────────┘
                         ▼
              ┌─────────────────────┐
              │    3x3 Conv         │   空间特征提取
              │    BatchNorm        │   out_channels → out_channels
              │    ReLU             │
              └──────────┬──────────┘
                         ▼
              ┌─────────────────────┐
              │    1x1 Conv         │   升维: out_channels → out_channels×4
              │    BatchNorm        │
              └──────────┬──────────┘
                         │
                         ▼
                        (+) ◄── 残差连接 (如果维度变化则经过 downsample)

───────────────────────────────────────────────────────────────────────────────
参数数量计算 (以 layer1 为例):
- conv1: 1×1×256×64 = 16,384
- conv2: 3×3×64×64 = 36,864
- conv3: 1×1×64×256 = 16,384
- BatchNorm: 3 × 256 = 768 (含 running_mean, running_var, weight, bias)
总计: ~70K 参数 per block
```

## 6. 训练超参数详解

### 6.1 模型相关参数

| 参数 | 默认值 | 说明 | 调参建议 |
|------|--------|------|----------|
| `--model` | ResNet50 | 模型变体 | 可选 ResNet50/101/152，越深越容易过拟合 |
| `--dataset` | CIFAR10 | 数据集 | CIFAR10 (10类) / CIFAR100 (100类) |

### 6.2 训练相关参数

| 参数 | 默认值 | 说明 | 调参建议 |
|------|--------|------|----------|
| `--epochs` | 200 | 训练轮数 | CIFAR 通常 200-300，配合余弦退火效果最佳 |
| `--batch-size` | 128 | 批大小 | GPU 显存允许下越大越好，128 是 CIFAR 的常用值 |
| `--lr` | 0.1 | 初始学习率 | SGD 配合 momentum=0.9 时，0.1 是标准初始值 |
| `--weight-decay` | 1e-4 | L2 正则化 | 防止过拟合，过大导致欠拟合 |
| `--momentum` | 0.9 | SGD 动量 | 标准值，加速收敛 |

### 6.3 正则化参数

| 参数 | 默认值 | 说明 | 调参建议 |
|------|--------|------|----------|
| `--label-smoothing` | 0.1 | 标签平滑 | 0.1-0.2，减少过拟合，提升泛化 |
| `--warmup-epochs` | 5 | 预热轮数 | 3-10，防止初期 loss 爆炸 |

### 6.4 其他参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--num-workers` | 4 | DataLoader 并行数 |
| `--seed` | 42 | 随机种子，保证可复现性 |
| `--data-dir` | ./data | 数据集存储路径 |
| `--checkpoint-dir` | ./checkpoints | 模型保存路径 |
| `--resume` | None | 从检查点恢复训练 |

## 7. 完整超参数对照表

```python
# 默认训练配置
DEFAULT_CONFIG = {
    # 模型配置
    "model": "ResNet50",           # ResNet50 / ResNet101 / ResNet152
    "num_classes": 10,             # CIFAR10=10, CIFAR100=100

    # 训练配置
    "epochs": 200,
    "batch_size": 128,
    "lr": 0.1,
    "weight_decay": 1e-4,
    "momentum": 0.9,
    "nesterov": True,              # Nesterov 动量加速

    # 学习率调度
    "warmup_epochs": 5,
    "scheduler_type": "warmup + cosine",

    # 正则化
    "label_smoothing": 0.1,

    # 数据配置
    "dataset": "CIFAR10",
    "num_workers": 4,

    # 随机性
    "seed": 42,
}

# CIFAR-10 数据增强配置
CIFAR10_TRAIN_TRANSFORMS = [
    transforms.RandomCrop(32, padding=4),      # 随机裁剪

        std=(0.2023, 0.1994, 0.2010)
    ),
    transforms.RandomErasing(p=0.1),           # 随机擦除
]

# CIFAR-100 数据增强配置
CIFAR100_MEAN = (0.5071, 0.4867, 0.4408)
CIFAR100_STD = (0.2675, 0.2565, 0.2761)
```

## 8. 梯度计算流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         单步训练流程 (GPU)                                  │
└─────────────────────────────────────────────────────────────────────────────┘

                         Batch Data (images, labels)
                                    │
                                    ▼
                    ┌───────────────────────────────┐
                    │     CUDA / Mixed Precision    │
                    │     (torch.amp.autocast)       │
                    └───────────────┬───────────────┘
                                    ▼
                    ┌───────────────────────────────┐
                    │      前向传播 (Forward)       │
                    │  output = model(images)       │
                    └───────────────┬───────────────┘
                                    ▼
                    ┌───────────────────────────────┐
                    │      损失计算 (Loss)          │
                    │  loss = criterion(output, labels) │
                    │  CrossEntropyLoss + Label Smoothing │
                    └───────────────┬───────────────┘
                                    ▼
                    ┌───────────────────────────────┐
                    │      清零梯度 (Zero Grad)     │
                    │  optimizer.zero_grad()        │
                    └───────────────┬───────────────┘
                                    ▼
                    ┌───────────────────────────────┐
                    │      反向传播 (Backward)       │
                    │  loss.backward()              │
                    │  scaler.scale(loss).backward() │
                    └───────────────┬───────────────┘
                                    ▼
                    ┌───────────────────────────────┐
                    │      梯度缩放 (Scale)         │
                    │  scaler.step(optimizer)        │
                    │  scaler.update()              │
                    └───────────────┬───────────────┘
                                    ▼
                                    │  更新模型参数
                                    ▼
                         梯度裁剪 (可选)
                         max_norm = 1.0
```

## 9. 检查点保存结构

```
checkpoints/
├── best.pth          # 验证集最佳模型
│   ├── epoch: int              # 训练轮数
│   ├── model_name: str          # "ResNet50"

│   ├── scheduler_state_dict     # 学习率调度器状态
│   ├── best_val_acc: float      # 最佳验证准确率
│   ├── args: dict               # 训练超参数
│   └── history: list            # 完整训练历史
│
└── last.pth          # 最后一个 epoch 的模型
    └── (同上)
```

## 10. 命令行使用示例

```bash
# 基本训练 (使用默认参数)
python train_resnet.py

# 自定义训练
python train_resnet.py \
    --model ResNet50 \
    --dataset CIFAR10 \
    --epochs 200 \
    --batch-size 128 \
    --lr 0.1 \
    --weight-decay 1e-4 \
    --label-smoothing 0.1 \
    --warmup-epochs 5

# CIFAR-100 训练
python train_resnet.py \
    --model ResNet50 \
    --dataset CIFAR100 \
    --epochs 300 \
    --batch-size 64

# 从检查点恢复训练
python train_resnet.py --resume ./checkpoints/last.pth

# ResNet101 训练
python train_resnet.py --model ResNet101 --epochs 150

# 指定数据目录
python train_resnet.py --data-dir /path/to/data --checkpoint-dir /path/to/checkpoints
```

## 12. ResNet50 vs ResNet101 vs ResNet152 对比

| 特性 | ResNet50 | ResNet101 | ResNet152 |
|------|----------|-----------|-----------|
| **Bottleneck 块数** | [3, 4, 6, 3] | [3, 4, 23, 3] | [3, 8, 36, 3] |
| **参数量** | 25.6M | 44.5M | 60.2M |
| **计算量 (GFLOPs)** | 4.1 | 7.8 | 11.5 |
| **CIFAR-10 准确率** | ~95% | ~96% | ~96% |
| **训练时间 (200 epochs)** | ~2.5h | ~4h | ~6h |
| **适用场景** | 资源有限 | 平衡性能 | 追求最高精度 |
