# MNIST AutoEncoder 详细流程图

本文档按照纯文本框图的形式，详细展示 `mnist_autoencoder.py` 的整体执行流程、模型构建流程、前向传播流程、训练流程以及张量尺寸变化过程。整体风格参考你给出的示意图，便于直接放入 Markdown 中查看。

---

## 1. `mnist_autoencoder.py` 总体主流程图

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│                         mnist_autoencoder.py 主流程                         │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段1：导入依赖模块                                                         │
│                                                                              │
│  导入 os                                                                     │
│  导入 typing.Tuple                                                           │
│  导入 torch / torch.nn / torch.nn.functional                                │
│  导入 Adam 优化器                                                            │
│  导入 StepLR 学习率调度器                                                    │
│  导入 DataLoader                                                             │
│  导入 torchvision.datasets / transforms                                     │
│  导入 save_image                                                             │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段2：定义网络基础模块                                                      │
│                                                                              │
│  定义 DownConvLayer                                                          │
│    ├─ Conv2d(dim -> dim, 3x3, padding=1)                                    │
│    ├─ ReLU                                                                   │
│    └─ MaxPool2d(2x2)                                                         │
│                                                                              │
│  定义 UpConvLayer                                                            │
│    ├─ Conv2d(dim -> dim, 3x3, padding=1)                                    │
│    ├─ ReLU                                                                   │
│    └─ Upsample(scale_factor=2, mode=nearest)                                │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段3：定义完整模型结构                                                      │
│                                                                              │
│  定义 Encoder                                                                │
│    └─ 使用 ModuleList 重复堆叠多个 DownConvLayer                            │
│                                                                              │
│  定义 Decoder                                                                │
│    ├─ 使用 ModuleList 重复堆叠多个 UpConvLayer                              │
│    ├─ final_conv = Conv2d(dim -> 1, 3x3, padding=1)                         │
│    └─ Sigmoid 输出到 0~1                                                     │
│                                                                              │
│  定义 AutoEncoderModel                                                       │
│    ├─ self.encoder = Encoder(dim=1, layer_num=2)                            │
│    └─ self.decoder = Decoder(dim=1, layer_num=2)                            │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段4：定义功能函数                                                          │
│                                                                              │
│  build_dataloader()                                                          │
│    ├─ Resize(32, 32)                                                         │
│    ├─ ToTensor()                                                             │
│    ├─ 加载 MNIST 训练集                                                      │
│    └─ 返回 DataLoader                                                        │
│                                                                              │
│  train_one_epoch()                                                           │
│    ├─ 逐 batch 训练                                                          │
│    ├─ 前向传播                                                               │
│    ├─ 计算 MSE 损失                                                          │
│    ├─ 反向传播                                                               │
│    └─ 返回本轮平均损失                                                       │
│                                                                              │
│  save_reconstruction_samples()                                               │
│    ├─ 取一批图片                                                             │
│    ├─ 模型重建                                                               │
│    ├─ 拼接原图和重建图                                                       │
│    └─ 保存 reconstruction.png                                                │
│                                                                              │
│  setup_seed()                                                                │
│    └─ 设置 CPU / GPU 随机种子                                                │
│                                                                              │
│  prepare_output_dirs()                                                       │
│    └─ 创建权重目录和图片目录                                                 │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段5：进入 main()                                                           │
│                                                                              │
│  设置超参数                                                                  │
│    batch_size = 128                                                          │
│    learning_rate = 1e-3                                                      │
│    num_epochs = 10                                                           │
│    seed = 42                                                                 │
│    layer_num = 2                                                             │
│    step_size = 5                                                             │
│    gamma = 0.5                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段6：训练准备                                                              │
│                                                                              │
│  setup_seed(seed)                                                            │
│  自动选择 device = cuda 或 cpu                                               │
│  train_loader = build_dataloader(batch_size)                                 │
│  model = AutoEncoderModel(layer_num=2).to(device)                            │
│  optimizer = Adam(model.parameters(), lr=1e-3)                               │
│  criterion = MSELoss()                                                       │
│  scheduler = StepLR(step_size=5, gamma=0.5)                                  │
│  weight_dir, image_dir = prepare_output_dirs()                               │
│  best_loss = inf                                                             │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段7：开始 epoch 循环                                                       │
│                                                                              │
│  for epoch in range(1, num_epochs + 1):                                      │
│    ├─ epoch_loss = train_one_epoch(...)                                      │
│    ├─ scheduler.step()                                                       │
│    ├─ 打印当前 loss 和 lr                                                    │
│    └─ 如果 epoch_loss < best_loss，则保存 best_autoencoder.pth              │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 阶段8：训练结束后的收尾处理                                                  │
│                                                                              │
│  保存 last_autoencoder.pth                                                   │
│  调用 save_reconstruction_samples()                                          │
│  保存 reconstruction.png                                                     │
│  打印训练完成信息                                                            │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│                                    结束                                      │
└──────────────────────────────────────────────────────────────────────────────┘
```

```text
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐

└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第2步：进入 AutoEncoderModel.__init__()                                      │
│                                                                              │

                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第3步：构建 Encoder                                                          │
│                                                                              │
│  进入 Encoder.__init__(dim=1, layer_num=2)                                  │

└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼

                                      │
                                      ▼

│                                                                              │
│  DownConvLayer(dim=1)                                                        │
│    ├─ super().__init__()                                                     │
│    ├─ self.conv = Conv2d(1, 1, kernel_size=3, padding=1)                    │
│    └─ self.pool = MaxPool2d(2, 2)                                            │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼

                                      │
                                      ▼

                                      │
```

```text
┌──────────────────────────────────────────────────────────────────────────────┐

                                      ▼

                                      │
                                      ▼

                                      │

                                      ▼

                                      │
```

## 4. `train_one_epoch()` 训练流程图

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│                           train_one_epoch() 训练流程                         │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第1步：model.train()                                                         │
│                                                                              │
│  将模型切换为训练模式                                                        │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第2步：初始化统计量                                                          │
│                                                                              │
│  running_loss = 0.0                                                          │
│  total_samples = 0                                                           │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第3步：遍历 DataLoader                                                       │
│                                                                              │
│  for images, _ in loader:                                                    │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第4步：处理当前 batch                                                        │
│                                                                              │
│  images = images.to(device)                                                  │
│  optimizer.zero_grad()                                                       │
│  reconstructed_images = model(images)                                        │
│  loss = criterion(reconstructed_images, images)                              │
│  loss.backward()                                                             │
│  optimizer.step()                                                            │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第5步：累计损失                                                              │
│                                                                              │
│  batch_size = images.size(0)                                                 │
│  running_loss += loss.item() * batch_size                                    │
│  total_samples += batch_size                                                 │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第6步：判断 batch 是否处理完                                                 │
│                                                                              │
│  若还有下一个 batch -> 返回第3步                                             │
│  若当前 epoch 所有 batch 已结束 -> 继续下一步                                │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 第7步：返回平均损失                                                          │
│                                                                              │
│  epoch_loss = running_loss / total_samples                                   │
└──────────────────────────────────────────────────────────────────────────────┘
```

```text

                                      │

                                      │

                                      │
                                      ▼

│                                                                              │
│  保存 last_autoencoder.pth                                                   │
```

```text
                                      │

┌──────────────────────────────────────────────────────────────────────────────┐
│ 解码器第2层 UpConvLayer                                                      │
│                                                                              │
│  [batch, 1, 16, 16] -> [batch, 1, 32, 32]                                   │
└──────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│ 最终输出 reconstruction                                                      │

```

## 7. 流程图配套说明

### 7.1 这份流程图重点回答了什么

这份文档重点回答了以下几个问题：

1. `mnist_autoencoder.py` 是按什么顺序执行的。
2. 自编码器模型是如何一步一步构建出来的。
3. 编码器和解码器在前向传播时分别做了什么。
4. 每一轮训练中具体执行了哪些步骤。
5. 输入图像在网络内部的尺寸是如何变化的。

### 7.2 为什么模型构建要单独画图

因为“训练流程”和“模型构建流程”是两个不同层次的问题：

- 训练流程回答的是：程序怎么跑。
- 模型构建流程回答的是：网络怎么搭。

把这两部分拆开画，可以更容易理解：

- `AutoEncoderModel` 是如何由 `Encoder` 和 `Decoder` 组成的。
- `Encoder` 和 `Decoder` 又是如何由多个基础模块堆叠出来的。

### 7.3 当前模型结构总结

当前脚本中模型结构可以概括为：

```text
输入图像
  -> Encoder
      -> DownConvLayer #1
      -> DownConvLayer #2
  -> latent 特征
  -> Decoder
      -> UpConvLayer #1
      -> UpConvLayer #2
      -> final_conv
  -> 重建图像
```
