## 3. VAE 前向传播流程图

```text
┌────────────────────────────────────────────────────────────────────────────┐
│                          VAE 前向传播 forward()                            │
└────────────────────────────────────────────────────────────────────────────┘
                                     │
                                     ▼
                      ┌──────────────────────────────┐
                      │ 输入图像 images               │
                      │ shape = [batch, 1, 28, 28]   │
                      └──────────────────────────────┘
                                     │
                                     ▼
                      ┌──────────────────────────────┐
                      │ 展平操作 flatten              │
                      │ shape = [batch, 784]          │
                      └──────────────────────────────┘
                                     │
                                     ▼
                      ┌──────────────────────────────┐
                      │ encode(x)                     │
                      │ x -> Linear(784,400) -> ReLU │
                      └──────────────────────────────┘
                                     │
                                     ▼
              ┌──────────────────────┴──────────────────────┐
              ▼                                             ▼
┌──────────────────────────────┐              ┌──────────────────────────────┐
│ fc_mu(x)                     │              │ fc_log_var(x)                │
│ 得到 mu                      │              │ 得到 log_var                 │
└──────────────────────────────┘              └──────────────────────────────┘
              │                                             │
              └──────────────────────┬──────────────────────┘
                                     ▼
                      ┌──────────────────────────────┐
                      │ reparameterize(mu, log_var)  │
                      │                              │
                      │ std = exp(0.5 * log_var)     │
                      │ eps = randn_like(std)        │
                      │ z = mu + eps * std           │
                      └──────────────────────────────┘
                                     │
                                     ▼
                      ┌──────────────────────────────┐
                      │ decode(z)                    │
                      │ z -> Linear(20,400) -> ReLU │
                      │   -> Linear(400,784)        │
                      │   -> Sigmoid                │
                      └──────────────────────────────┘
                                     │
                                     ▼
                      ┌──────────────────────────────┐
                      │ reconstructed                │
                      │ shape = [batch, 784]         │
                      └──────────────────────────────┘
```

## 8. VAE 核心原理图补充

下面这张补充图对应你发来的教材示意图，重点突出 VAE 与普通自编码器不同的地方：编码器不是直接输出固定特征，而是输出均值和方差，用来定义潜变量分布，再从该分布中采样得到 `z`，最后交给解码器生成结果。

```text
┌──────────┐      ┌──────────┐
│ 原始数据  │ ───▶ │  编码器   │
└──────────┘      └──────────┘
                        │
            ┌───────────┴───────────┐
            ▼                       ▼
      ┌──────────┐            ┌──────────┐
      │   均值μ   │            │   方差σ²  │
      └──────────┘            └──────────┘
            │                       │
            └───────────┬───────────┘
                        ▼
                 ┌──────────────┐
                 │  定义正态分布  │
                 │ N(μ, σ²)      │
                 └──────────────┘
                        │
                        │   与标准正态分布比较
                        │   计算分布一致 Loss
                        ▼
                 ┌──────────────┐
                 │ 标准正态分布  │
                 │   N(0, 1)    │
                 └──────────────┘

原始数据 ───────────────────────────────────────────────┐
                                                       │
                                                       ▼
                                                   重建 Loss
                                                       ▲
                                                       │
                 ┌──────────────┐     采样    ┌──────┐ │   ┌──────────┐   ┌──────────┐
                 │  正态分布     │ ─────────▶ │  z   │ ───▶ │  解码器   │ ─▶ │ 重建数据  │
                 │ N(μ, σ²)      │            └──────┘     └──────────┘   └──────────┘
                 └──────────────┘
```

### 8.1 图中各部分与代码的对应关系

- 图中的“编码器”对应 `VAEModel.encode()`
- 图中的“均值 μ”对应 `fc_mu`
- 图中的“方差/对数方差”对应 `fc_log_var`
- 图中的“采样”对应 `reparameterize(mu, log_var)`
- 图中的“z”对应潜变量 `z`
- 图中的“解码器”对应 `VAEModel.decode()`
- 图中的“重建 Loss”对应 `recon_loss`
- 图中的“分布一致 Loss”对应代码中的 `kl_divergence`

### 8.2 为什么图里是“方差”，代码里是 `log_var`

教材图一般写成方差 `σ²`，更直观；而代码中通常预测 `log_var`，原因是：

- 直接预测方差容易出现数值不稳定
- 预测对数方差更适合神经网络优化
- 计算标准差时再通过 `std = exp(0.5 * log_var)` 还原

### 8.3 这张图对应的核心公式

VAE 的核心采样过程可以写成：

```text
z = μ + ε * σ,   其中 ε ~ N(0, 1)
```

其中：

- `μ` 由编码器预测
- `σ` 由方差或对数方差换算得到
- `ε` 来自标准正态分布

这样就把原本不可导的“直接采样”转化成了可导的运算组合，也就是重参数化技巧。

## 9. 简要说明

这份流程图重点展示了 VAE 相比普通 AE 多出来的关键步骤：

- 编码器不直接输出固定特征，而是输出 `mu` 和 `log_var`
- 通过重参数化技巧从潜变量分布中采样 `z`
- 损失函数不仅包含重建损失，还包含 `KL` 散度
- 训练结束后除了保存重建图，还会随机采样生成新图像
