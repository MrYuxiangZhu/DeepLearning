## 4. OWLv2 检测模型原理

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           OWLv2 零样本目标检测器                                │
│                    Open-World Learning v2 Zero-Shot Detector                    │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              模型架构                                           │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │                          文本输入                                        │   │
│  │   ["person", "dog", "cat", "tiger", "horse", "sheep", ...]             │   │
│  │                               │                                          │   │
│  │                               ▼                                          │   │
│  │   ┌─────────────────────────────────────────────────────────────────┐  │   │
│  │   │                    Text Encoder (Transformer)                    │  │   │
│  │   │                                                                 │  │   │
│  │   │   Text → Tokenize → Embedding → Transformer Layers → Text Feat  │  │   │
│  │   │   [CLS] token 作为文本全局特征                                    │  │   │
│  │   └─────────────────────────────────────────────────────────────────┘  │   │
│  │                               │                                          │   │
│  │                               ▼                                          │   │
│  │   ┌─────────────────────────────────────────────────────────────────┐  │   │
│  │   │                    Image Encoder (Vision Transformer)            │  │   │
│  │   │                                                                 │  │   │
│  │   │   Image → Patch Embed → 16×16 Patches → ViT Encoder            │  │   │
│  │   │   输出: 图像补丁特征 + 全局 [CLS] 特征                           │  │   │
│  │   └─────────────────────────────────────────────────────────────────┘  │   │
│  │                               │                                          │   │
│  │                               ▼                                          │   │
│  │   ┌─────────────────────────────────────────────────────────────────┐  │   │
│  │   │                    Feature Fusion Layer                          │  │   │
│  │   │                                                                 │  │   │
│  │   │   图像特征与文本特征进行跨模态注意力融合                          │  │   │
│  │   │   Image_Feature_Aligned_With_Text                               │  │   │
│  │   └─────────────────────────────────────────────────────────────────┘  │   │
│  │                               │                                          │   │
│  │                               ▼                                          │   │
│  │   ┌─────────────────────────────────────────────────────────────────┐  │   │
│  │   │                    Detection Head                                │  │   │
│  │   │                                                                 │  │   │
│  │   │   - 边界框回归: (x, y, w, h) 预测                               │  │   │
│  │   │   - 类别置信度: 每个文本类别的匹配分数                           │  │   │
│  │   └─────────────────────────────────────────────────────────────────┘  │   │
│  │                               │                                          │   │
│  │                               ▼                                          │   │
│  │                        检测结果输出                                     │   │
│  │   [(box, score, label), ...]                                          │   │
│  │                                                                 │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           OWLv2 损失函数                                         │
└─────────────────────────────────────────────────────────────────────────────────┘

总损失 = 分类损失 + 边界框损失

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              分类损失 (Focal Loss)                              │
│                                                                                 │
│  L_cls = -α × (1 - p_t)^γ × log(p_t)                                         │
│                                                                                 │
│  其中:                                                                            │
│  - p_t: 预测框与 GT 框的 IoU                                                    │
│  - α: 平衡因子 (默认 0.25)                                                     │
│  - γ: 聚焦因子 (默认 2.0)                                                      │
│                                                                                 │
│  作用: 降低易分类样本权重，聚焦困难样本                                          │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                            边界框损失 (L1 + GIoU)                               │
│                                                                                 │
│  L_box = λ_L1 × L1_loss + λ_GIoU × GIoU_loss                                  │
│                                                                                 │
│  L1_loss = Σ |pred_box - gt_box|                                              │
│                                                                                 │
│  GIoU = IoU - (C - (A ∪ B)) / C                                               │
│  其中 C 是包围 pred 和 gt 的最小矩形                                            │
└─────────────────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              CLIP 图像-文本匹配模型                               │
│                   Contrastive Language-Image Pre-training                        │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              模型架构                                           │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                         │   │
│  │   图像输入                           文本输入                           │   │
│  │       │                                   │                             │   │
│  │       ▼                                   ▼                             │   │
│  │   ┌─────────┐                       ┌─────────┐                        │   │
│  │   │   ViT   │                       │  Text   │                        │   │
│  │   │ Encoder │                       │ Encoder │                        │   │
│  │   └────┬────┘                       └────┬────┘                        │   │
│  │        │                                 │                             │   │
│  │        ▼                                 ▼                             │   │
│  │   I_fea [D]                       T_fea [D]                          │   │
│  │        │                                 │                             │   │
│  │        ▼                                 ▼                             │   │
│  │   ┌─────────┐                       ┌─────────┐                        │   │
│  │   │Proj(W_I)│                       │Proj(W_T)│                       │   │
│  │   └────┬────┘                       └────┬────┘                        │   │
│  │        │                                 │                             │   │
│  │        ▼                                 ▼                             │   │
│  │   I_emb [D]                       T_emb [D]                          │   │
│  │        │                                 │                             │   │
│  │        └──────────┬──────────────────────┘                             │   │
│  │                   │                                                       │   │
│  │                   ▼                                                       │   │
│  │         ┌─────────────────┐                                             │   │
│  │         │  对比学习损失    │                                             │   │
│  │         │   InfoNCE       │                                             │   │
│  │         └─────────────────┘                                             │   │
│  │                                                                         │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           相似度计算公式                                         │
└─────────────────────────────────────────────────────────────────────────────────┘

                       I_emb · T_emb
sim(I, T) = ─────────────────────────
            ||I_emb|| × ||T_emb||

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           Softmax 分类概率                                      │
│                                                                                 │
│                              exp(sim(I, T_k))                                  │
│  P(label=k) = ─────────────────────────────────                               │
│                Σ_j exp(sim(I, T_j))                                           │
│                                                                                 │
│  其中:                                                                            │
│  - I: 当前人物裁剪图像                                                           │
│  - T_k: 文本 "man" 或 "woman" 的嵌入                                            │
│  - 分母: 对所有候选文本的概率归一化                                               │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           CLIP 训练目标                                         │
│                                                                                 │
│  最大化匹配对 (图像 I_i, 文本 T_i) 的相似度                                      │
│  最小化不匹配对的相似度                                                          │
│                                                                                 │
│  L = -1/2 × [log(exp(sim(I_i, T_i))/Σ exp(sim(I_i, T_j)))                    │
│               + log(exp(sim(T_i, I_i))/Σ exp(sim(T_i, I_j)))]                │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 6. NMS 非极大值抑制

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                          NMS 非极大值抑制算法                                    │
│                        Non-Maximum Suppression                                   │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              算法流程                                           │
│                                                                                 │
│  输入: 检测框列表 D = {d1, d2, ..., dn}                                        │
│  输出: 过滤后的检测框列表 D'                                                    │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                         │   │
│  │  1. 按置信度降序排序: sort(D, key=score, reverse=True)                 │   │
│  │         ↓                                                              │   │
│  │  ┌────────────────────────────────────────────────────────────────┐  │   │
│  │  │  d_max ← pop(排序列表第一个)  ← 当前最高分框                    │  │   │
│  │  │  kept ← [d_max]                                                │  │   │
│  │  │  remaining ← []                                                │  │   │
│  │  │                                                                  │  │   │
│  │  │  for d in remaining_boxes:                                     │  │   │
│  │  │      if same_label(d, d_max) AND IoU(d, d_max) > threshold:     │  │   │
│  │  │          skip d  ← 与高分框同类且重叠，跳过                      │  │   │
│  │  │      else:                                                      │  │   │
│  │  │          remaining ← [d]  ← 保留                                 │  │   │
│  │  │                                                                  │  │   │
│  │  │  repeat until 列表为空                                          │  │   │
│  │  └────────────────────────────────────────────────────────────────┘  │   │
│  │         ↓                                                              │   │
│  │  return kept                                                           │   │
│  │                                                                         │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              IoU 计算公式                                        │
│                                                                                 │
│                              ┌──────────────────┐                               │
│                              │     Box A       │                               │
│                              │    ┌────────┐   │                               │
│                              │    │交集区域│   │                               │
│                              │    │┌──────┴┐  │                               │
│                              │    ││ Box B │  │                               │
│                              │    │└──────┘  │                               │
│                              │    └──────────┘   │                               │
│                              └──────────────────┘                               │
│                                                                                 │
│              Area(A ∩ B)                                                        │
│  IoU = ──────────────────                                                     │
│        Area(A ∪ B)                                                              │
│                                                                                 │
│  其中:                                                                            │
│  - Area(A ∩ B) = max(0, x2-x1) × max(0, y2-y1)                                │
│  - Area(A ∪ B) = Area(A) + Area(B) - Area(A ∩ B)                              │
│                                                                                 │
│  IoU ∈ [0, 1]:                                                                 │
│  - IoU = 0: 完全不相交                                                         │
│  - IoU = 1: 完全重叠                                                           │
│  - IoU > 0.5: 高重叠                                                           │
│  - IoU > 0.7: 严重重叠，需要抑制                                                │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              NMS 效果示意                                       │
│                                                                                 │
│   原始检测框                         NMS 处理后                                │
│                                                                                 │
│      ┌─────┐ ┌─────┐              ┌─────┐                                    │
│      │ 0.9 │ │ 0.7 │              │ 0.9 │                                    │
│      │     │ └─────┘              └─────┘                                    │
│      │ 0.6 │  ← 重叠框 → (IoU>0.4) 抑制                                      │
│      └─────┘                                                                   │
│                                                                                 │
│   [保留 0.9, 0.6]              [保留 0.9, 0.85]                               │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 11. 核心公式汇总

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              核心公式汇总                                       │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                              IoU 计算                                           │
│                                                                                 │
│                               max(x1_a, x1_b)                                  │
│  x1 = ────────────────────────────────                                        │
│       max(x2_a, x2_b) - min(x1_a, x1_b)                                        │
│                                                                                 │
│  IoU = Area(A ∩ B) / Area(A ∪ B)                                              │
│      = (w × h) / (Area(A) + Area(B) - w × h)                                  │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           CLIP 相似度                                           │
│                                                                                 │
│                        I_emb · T_emb                                          │
│         sim = ─────────────────────────────                                   │
│                ||I_emb|| × ||T_emb||                                          │
│                                                                                 │
│                exp(sim(I, T_k))                                                │
│         P_k = ───────────────────────────                                      │
│                Σ_j exp(sim(I, T_j))                                           │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────┐
│                           综合置信度                                            │
│                                                                                 │
│         final_score = detection_score × gender_score                          │
│                                                                                 │
│         只有当 gender_score ≥ 0.5 时才进行细分类                                  │
│         否则保持原始 "person" 标签                                              │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```
